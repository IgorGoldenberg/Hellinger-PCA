# Hellinger-PCA
PCA-Based Drift and Shift Quantification Framework for Multidimensional Data
This repository provides scripts created while working on the above paper, published in
Knowledge and Information Systems (2020) 62:2835–2854
https://doi.org/10.1007/s10115-020-01438-3
HDTolerance.R contains AnalysisOfZeroHD function, used for calibration, described in Section 5
Hellingerdistance.R contains FullHDPCAExperiment function, used in Section 6
There are also numerous helper functions in both files
